**js/nav.js creation:**

Create a new file `js/nav.js` with an IIFE that:
1. Gets `#nav-hamburger` and `#nav-mobile-panel` by ID
2. On hamburger click: toggles `is-open` class on panel, sets `aria-expanded` true/false
3. On document click: if click target is not inside `#site-nav` and not inside `#nav-mobile-panel`, remove `is-open` and set `aria-expanded` to false
4. On any `.nav-mobile-panel a` click: close the panel (so navigation closes the menu)

This is an external JS file so it requires NO CSP hash changes. Pages just add `<script src="js/nav.js"></script>`.

**Important:** Do NOT remove any existing CSS classes that are used by page-specific content (post styles, card styles, etc.). Only modify/replace the header, navigation, footer, and responsive nav sections.
